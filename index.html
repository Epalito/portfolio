Site Statique

Projet de Site Statique Personnel :

• Objectif : Reproduire un site connu en utilisant mes compétences en HTML/CSS acquises au cours de mon auto-formation, tout en le faisant évoluer progressivement avec des ajouts divers.
• Les différents éléments du site sont interactifs, avec des effets de survol et un carrousel fonctionnel offrant à la fois un switch manuel et automatique.
• Les logos et les images sont des liens redirigeant vers des pages officielles, ajoutant une dimension fonctionnelle au site.
• Le site est conçu de manière à être responsive, garantissant une expérience utilisateur optimale sur différentes tailles d'écrans. Ce projet m'a permis d'appliquer concrètement mes connaissances en développement web et de renforcer mes compétences en conception de sites interactifs et adaptatifs.

Création d'une API Rest

Ce projet vise à approfondir mes compétences dans l'utilisation de NodeJS en développant une API REST. Au cours de ce projet, j'ai utilisé divers composants pour construire l'API, notamment des middlewares, Express, Nodemon, Sequelize (ORM), Bcrypt et Jsonwebtoken.

Outils utilisés pendant le développement :
• Insomnia pour les tests d'API
• XAMPP et phpMyAdmin pour la gestion de la base de données
• Heroku/Heroku CLI/Git pour le déploiement

En plus d'acquérir une expérience pratique avec ces outils, j'ai également exploré d'autres concepts tels que :
• Gestion des erreurs : résolution des erreurs techniques (mise en place de catch, clarification des messages pour les utilisateurs), gestion des erreurs métier (utilisation de validateurs, contraintes).
• Requêtes avancées
• Authentification avec les jetons JWT
• Déploiement : ajustement des variables d'environnement pour la production, optimisation des dépendances.

Site e-commerce

Contexte : Dans le cadre de ma progression en tant que développeur web, j'ai entrepris la création d'un site e-commerce basé sur un modèle défini. Ce projet ambitieux vise à consolider mes connaissances en HTML, CSS, et JavaScript tout en explorant les technologies avancées du développement web.

Compétences acquises :
• JavaScript Intermédiaire : Maîtrise du DOM, utilisation de jsrender pour la gestion dynamique du contenu, et implémentation d'un système de routage d'URLs.
• JavaScript Avancé : Application de concepts asynchrones, adoption des fonctionnalités d'ES6 pour optimiser le code.
• NodeJS : Développement d'une Web API robuste, intégration du module Body Parser, gestion du chargement de fichiers, et utilisation de jetons JWT pour la sécurité.
• MongoDB : Conception d'une base de données solide, utilisation du framework Mongoose, implémentation de filtres pour une expérience utilisateur améliorée.
• Outils de Développement : Utilisation d'ESLint et Babel pour maintenir un code propre et compatible, gestion de versions avec Git et Github, éditeur de code Visual Studio Code.
• Déploiement : La phase de déploiement n'est pas encore définie à ce stade, mais sera envisagée au fur et à mesure du développement.
Statut : Actuellement en cours de développement.
